# Wake-up-Friends

## Desc
- AWS Console 들어가서 EC2 인스턴스를 Running / Stopped 하기 귀찮음
- 그냥 보고 깨우자 (Running)

## Exec 

```sh
    ## for dev
    make dev

    ## for prod
    make run
```
